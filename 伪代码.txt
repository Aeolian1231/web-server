main:
初始化日志记录模块
while 
    if  有客户端建立了连接
        设置超时5s
    else
        无阻塞
    select：阻塞等待，直到新的请求
    if  ret == -1
        错误处理
    /*  else if  ret == 0
            超时，但是不在此进行处理，因为在有新的连接或新的请求时ret > 0，而某些连接可能已经超时，但不会进入此分支
    */
    else if ret > 0
        if  FD_ISSET：检查是否有新的连接
            在clients结构体记录IP、port和连接时间
        FD_SET：将客户端套接字加入监控区
        for 遍历所有建立的连接
            if  有新的请求
                recv：接受请求
                if  接受空值
                    关闭连接，从监控区移除
                else
                    Send_back：传递到处理函数返回
    for 遍历所有建立的连接
        if  连接时间>5s
            超时处理，断开连接
关闭服务端
